Willkommen zum Webservice zur Datenabfrage von Filmen mit Drehort in San Francisco.

Mit diesem Service können die Filmnamen und genauen Drehorte von Filmen aus San Francisco abgefragt werden.

Durch klicken auf die folgenden Links können die Daten im JSON Format abgefragt werden, entweder alle Daten auf einmal oder mit einem Filter nach "title" und "locations".

  • Gefilterte Daten nach Film
